## 2. 测试维度说明

| 维度 | 说明 |
|------|------|
| 功能正确性 | 状态码、响应体结构、数据匹配 |
| 数据完整性 | 必填字段存在、数据类型正确 |
| 边界条件 | 不存在的ID、空数据、极大ID |
| 异常处理 | 无效请求、404 响应 |
| 嵌套资源 | 子资源路由（/posts/1/comments 等） |
| 过滤查询 | postId / userId / albumId 过滤 |
| 分页查询 | _start/_limit、_page/_limit 分页 |
